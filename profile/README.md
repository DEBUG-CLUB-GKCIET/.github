Official GitHub organization profile for **DEBUG CLUB**, the Computer Science Club of **Ghani Khan Choudhury Institute of Engineering and Technology (GKCIET), Malda**.

## About
**DEBUG CLUB** is a student-driven community for learning, building, and sharing Computer Science—through coding sessions, peer learning, projects, and competitive programming.

**What we do**
- Build practical projects (web, app, systems, ML—depending on interest)
- Learn collaboratively via workshops, study jams, and mentorship
- Prepare for contests and interviews (DSA, CP, fundamentals)
- Contribute to open-source and maintain club repositories
